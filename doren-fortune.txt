 1. What is Semantic HTML? 
	Semantic HTML is the markeup that gives meaning to HTML code.
    2. What is HTML used for? 
	HTML is used as a foundation for the creation of websites.
    3. What is an attribute and where do we put it? 
	An HTML attribute provides extra information about an element. We put it within the first tag.
    4. What is the h1 tag used for? How many times should I use it on a page?
	The h1 tag is used to provide meaning to the search engine. You should only use it once, to define the content as the most important in the page.
    5. Name two tags that have required attributes
	<a> tag and <img> tag
    6. What do we put in the head of our HTML document? 
	<header>
    7. What is an id? 
	It is an attribute that specifies a unique ID for only one HTML element.
    8. What elements can I add an id to? 
	You can add elements to an <h1> element or even an <a href> element
    9. How many times can I use the same id on a page? 
	Only once.
    10. What is a class? 
	A class is used to specifically identify an HTML element, but can be used for multiple HTML elements.
    11. What elements can I add a class to? 
	<h> elements
    12. How many times can I use the same class on a page? 
	You can use it multiple times.
    13. How do I get my link to open in a new tab?
	You can use target="_blank" within the <a href> to get a link to open in a new tag.
    14. What is the alt attribute used for? 
	Used to provide alternative information about an image.
    15. How do I reference an id?
	#
    16. What is the difference between a section and a div?
	Section groups content together in one and defines them--it is very distinct. A div does not have any specific meaning, but is used to divide the pages.
    17. What is CSS used for? 
	CSS is used to style a page.
    18. How to we select an element? Example - every h2 on the page
	h2{}
    19. What is the difference between a class and an id? - Give me an example of when I might use each one
	A class can be applied to multiple elements. An ID can only be applied to one. You might use a class if you want to change the color of several elements at once, and you might use an ID if you want that specific element to be the only one with changes.

    20. How do we select classes in CSS?
	.class {}
    21. How do we select a p element with a single class of “human””?
	.human{}
    22. What is a parent child selector? When would this be useful? 
	Parent child selector is when the style straits from a ‘parent’ element pass down to the generations after it, or the ‘child.’ This would be useful when you want to style all the elements after one element in a web page.
    23. How do you select all links within a div with the class of sidebar?
	div sidebar{}
    24. What is a pseudo selector?
	Used to define a special element state; change colors when user hovers, changes colors of links when visited or not, etc.
    25. What do we use the change the spacing between lines?
	line-height
    26. What do we use to change the spacing between letters?
	letter-spacing
    27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?

	text-transform:capitalize;
    28. How do I add a 1px border around my div that is dotted and black?
	border-style: dotted;
    29. How do I select everything on the page?

    30. How do I write a comment in CSS?

    31. How do I find out what file I am in, when I am using the command line? 
	pwd
    32. Using the command line - how do I see a list of files/folders in my current folder?
